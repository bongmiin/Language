### try-throw-catch
	
- try 블록
- throw {}
	- 반드시 tr y블록 안에 있어야 함
- catch {}
	- throw에 의해 발생한 예외를 처리하는 코드

> 0으로 나누는 예외 처리

> 지수 승 계산을 예외 처리 코드로 재작성

### throw 사용 주의 사항

- throw의 위치가 항상 try 블록 안에 있어야 한다
- 예외를 처리할 catch가 없으면 프로그램 강제 종료
- catch 블록 내에서도 try catch() {} 블록 선언 가능
